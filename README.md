# 🔋 Visualization Tool For Electric Vehicle Charge And Range Analysis


# EV Visualization Tool

A web-based visualization tool for analyzing electric vehicle (EV) charge data, energy consumption, and driving range. This project helps users explore EV performance and charging patterns through interactive charts and graphs.

---

## 🚀 Features

- Import EV charge and range data from CSV files
- Plot:
  - Charge sessions over time
  - Energy consumption trends
  - Distance and range analysis
- Interactive graphs using Plotly
- Responsive web interface
- Export visualizations as images
